# 亚马逊的基础关键词

# 1.Listing

指亚马逊网站上的商品详情页面，每一款商品通过卖家后台上传之后，就会生成一个独立的listing页面。买家可以通过listing页面来浏览和购买商品
Listing也是产品的流量承载页面，不管是站内还是站外流量都需要通过listing页面进行承载，进而形成转化。listing页面包含产品标题、价格、图片、五点描述、长描述/A+、QA...

# 2.ASIN

全称Amazon standard identifcation number，亚马逊标准识别号，在产品发布之后，亚马逊会随机生成一串10位由数字和字母所组成的编码，通常以B0为开头。这串编码具有唯一性，就好比产品的身份证
作用:定位产品(设置优惠促销、广告、反查竞品...)

# 3.FBA

指卖家将商品批量发送至亚马逊运营中心，由亚马逊负责帮助卖家存储，当商品售出后，由亚马逊完成订单分拣、包装和配送，并为这些商品提供买家咨询、退货等客户服务，帮助卖家节省大量的人力、物力和财力
头程物流：从国内配送至亚马逊仓库的运输流程
尾程物流：产生订单后，亚马逊配送订单的运输流程

# 4.FBM

指卖家自己配送在亚马逊销售的订单，并且从库存管理、包装、配送、客户服务等一系列流程都由卖家自行负责。从收到订单到配送给买家所经历的整个过程，每一个环节都由卖家自己把控
可以理解为亚马逊上除了FBA以外，其他的配送方式都算是FBM，包括第三方海外仓配送

# 5.跟卖

不同卖家之间的同款产品共用同一个详情页面，这也是亚马逊所特有的销售机制。为了给到客户更好的购物体验，把页面设置为一个产品、一个款式、一个品牌共享这个页面

# 6.商品编码

商品编码称为GTIN，包括UPC、EAN、JAN、ISBN。发布商品时需要填写的是UPC、EAN、JAN码
GTIN：全球贸易项目代码，包含UPC、EAN、JAN、ISBN
UPC：美国统一代码委员会制定的一种商品用条码，主要用于美国和加拿大地区
EAN：国际物品编码协会制定的一种商品用条码，通用于全世界，主要用于欧洲
JAN：日本工业标准(JIS)制定，主要用于亚马逊日本站
ISBN：国际标准书号，专门为识别图书等文献而设计的国际编号
GCID：全球目录编码，完成品牌备案后，系统进行分配

# 7.变体/父体/子体

变体：又称父/子关系，父体和子体这两个内容的一个统称。指彼此关联的一组商品，良好的变体关系体现在，可让买家根据不同的属性(包括尺寸、颜色或其他特性)通过商品商品详情页面上提供的选项比较和选择商品
